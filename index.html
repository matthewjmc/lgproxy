Welcome to the LGProxy Documentation!

Introduction

LGProxy is a high-performance proxy for the Looking Glass application. It utilizes the Telescope Remote Framebuffer Library for transport over both commodity and RDMA capable networking hardware.

View our GitHub repository.

Developed by the Telescope Project Team.
